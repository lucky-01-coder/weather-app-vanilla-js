# 🌦️ ऋतुDarpan

> A beautiful and responsive weather application with dynamic backgrounds, smooth animations, frosted glass UI, and real-time weather information.

---

## 📖 About

ऋतुDarpan is a modern weather application built using **HTML, CSS and Vanilla JavaScript**.

It provides real-time weather information using the **OpenWeather API** while delivering an immersive user experience through animated weather effects, dynamic backgrounds, and a clean glassmorphism design.

---

## ✨ Features

- 🌍 Search weather by city
- 📍 Get weather using current location
- 🌡️ Real-time temperature
- 🤗 Feels Like temperature
- 💧 Humidity
- 🌬️ Wind Speed
- 👀 Visibility
- 📈 Pressure
- 🌅 Sunrise & Sunset
- 📅 Local date & time
- 🌤️ Dynamic weather icons
- 🎨 Dynamic animated backgrounds
- ☁️ Animated moving clouds
- 🌧️ Rain animation
- 🧊 Frosted Glass UI
- 📱 Fully Responsive Design
- 💾 Last searched city saved using Local Storage
- 🔔 Toast notifications
- ⚡ Loading indicators

---

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/lucky-01-coder/ritudarpan.git
```

Open the project folder.

Create a `config.js` file.

```javascript
const CONFIG = {
  API_KEY: "YOUR_OPENWEATHER_API_KEY",
};
```

Now simply open **index.html** in your browser.
